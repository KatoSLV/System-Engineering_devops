# 0x00-shell_basics

This folder contains exercises focused on basic Linux commands, navigation, and file management.
It covers common tasks for working with the Linux command line, including file and directory operations.

## 📂 Structure
|File | Script Description |
|:----|:-------------------|
|0-Print_working_directory	| Exercise for pwd command – print the current working directory |
|1-List_files	| Exercise for ls command – list files and directories | 
|2-Change_directory	| Exercise for cd command – navigate between directories | 
|3-Create_file	| Exercise for touch command – create empty files |
|4-Copy_file	| Exercise for cp command – copy files |
|5-Move_file	| Exercise for mv command – move or rename files |
|6-Remove_file	| Exercise for rm command – delete files |
|7-Create_directory	| Exercise for mkdir command – create directories |
|8-Remove_directory	| Exercise for rmdir command – remove directories |
|9-List_files_of_the_following_destinations | List files of specified destinations |
|10-Create_a_file_and_check_the_type| Create a file and check its type |
|11-Create_a_symbolic_link_of_ls | Create a symbolic link to the ls command |
|12-Copy_all_files_with_a_specific_extension | Copy files with a specific extension |
|13-Move_all_files_containing_an_uppercase | Move files containing uppercase letters |
|14-Create_multiple_empty_files_with_extensions_ending_with_~ | Create multiple empty files with extensions ending in ~ |
|15-Create_the_following_directories_c_is_fun | Create multiple directories with a specific pattern |
|16-List_all_files_separated_by_commas | List all files separated by commas |

All scripts are written in Bash and can be run in any Linux or WSL environment.

## Learning Objectives

- Master basic Linux commands and file navigation

- Understand file types and basic file attributes

- Practice creating, moving, copying, and deleting files and directories

- Gain familiarity with shell operations and command-line workflows
