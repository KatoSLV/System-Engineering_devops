# ⚙️ System Engineering & DevOps

Repository with exercises and projects focused on System Engineering and DevOps fundamentals.

Main focus:

- Linux basics & shell

- Scripting & automation

- Networking fundamentals

- System processes & users

- Basic infrastructure concepts

## 📂 Structure
| Folder	| Content |
|:--------|:--------|
|0x00-shell_basics	| Linux commands, navigation, and file management |
|0x01-shell_permissions	| File permissions, users, groups |
|0x02-shell_redirections	| I/O redirection and pipes |
|0x03-variables_expansions	| Environment variables, expansions |

## Learning Goals

- Master **Linux commands** and **file management**

- Understand **file permissions**, **users**, and **groups**

- Learn **I/O redirection**, **pipes**, and **command chaining**

- Understand **environment variables** and **shell expansions**
